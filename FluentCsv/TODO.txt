== Options

- option pour faire désactiver le trim sur les données (le faire par défaut)
- option pour désactiver le trim sur les nom de colonnes (le faire par defaut)
- option pour ne pas prendre en compte les lignes vides

== Erreurs à gérer

- s'il y a plus de colonnes qu'il y en a dans la première ligne
- si une colonne pointe sur un index en dehors du nombe de colonne
- si on défini deux foix le même index (je crois que c'est déjà fait)

== Fonctionnalités

- gérer le RFC avec les doubles quotes
- possibilité de commenter des lignes en ajoutant un caractère (par defaut #)
- resultat avec liste des erreurs et resultset
- lire depuis un fichier
- lire depuis une ressource d'assembly
- Gestion de l'encoding
- Gestion de la culture
